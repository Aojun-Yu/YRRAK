# Assets

This folder is reserved for visual assets used by the Swing GUI.

Recommended file names:

```text
assets/
├── icons/
│   ├── fire.png
│   ├── water.png
│   └── thunder.png
├── characters/
│   ├── player.png
│   ├── training_dummy.png
│   ├── fire_spirit.png
│   └── thunder_beast.png
└── ui/
    ├── background.png
    └── card_frame.png
```

The game can run without these images. If an asset is missing, the GUI shows a colored placeholder instead.

When adding third-party or AI-assisted art, record the source and license in the README or development log.

The current PNG files are simple generated placeholder art created for the prototype. They are meant to make the GUI presentable while leaving room for better original art later.
