# Portfolio Summary

YRRAK is a Java console card battle game prototype created as a portfolio project.

## One-Sentence Description

YRRAK is an element-based card battle game where the player manages energy, block, and deck cycling to defeat three enemies with different behavior patterns.

## What This Project Demonstrates

- Java object-oriented programming.
- Game system design.
- Turn-based combat logic.
- Deck, hand, and discard pile management.
- Element advantage rules.
- Enemy intent and behavior patterns.
- In-game rules and final run summary.
- Clear separation between game logic and console input/output.
- Development documentation through README and DEVLOG files.

## Why It Matters

This project shows the process of building a complete small game system from simple classes into a playable MVP. It is intentionally scoped to be finished and understandable, with room to expand into a GUI version later.

## Current Status

Playable console MVP.
