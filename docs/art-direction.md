# Art Direction

## Visual Goal

YRRAK should look like a compact elemental card battle game, not a large fantasy RPG.

Recommended style:

- Dark fantasy card table
- Clean elemental colors
- Simple readable icons
- Slightly magical but not overly detailed
- UI first, character art second

## Color Language

- Fire: orange red
- Water: blue
- Thunder: yellow
- Background: dark navy
- UI panels: deep blue gray
- Text: soft white

## Required Assets

Minimum useful asset set:

```text
assets/icons/fire.png
assets/icons/water.png
assets/icons/thunder.png
assets/characters/player.png
assets/characters/training_dummy.png
assets/characters/fire_spirit.png
assets/characters/thunder_beast.png
assets/ui/background.png
```

## Asset Rules

- Keep all assets original, self-made, public domain, or properly licensed.
- If AI-assisted art is used, record that in the README or DEVLOG.
- Prefer readable icons over complex illustrations.
- Use consistent image proportions.
- Do not add too many assets before the core UI feels good.
